# nodejs-rest-example
> nodejs rest api example service

##
### Prerequisites
- node
  - https://nodejs.org/ko/
- mongodb
  - https://docs.mongodb.com/manual/installation/
  
---

### node modules install
```
npm install

  : package-json에 추가
    - scripts add
    "start": "nodemon"
```
> express, nodemon, morgan (logging), body-parser, mongoose, multer (images upload), bcrypt, jsonwebtoken

##
### RUN
npm start (서버실행)
